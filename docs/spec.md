## 1. ゲーム概要

放置型ダンジョン探索ゲーム。パーティを編成してダンジョンに送り出し、自動で探索・戦闘が進む。
ブラウザを閉じても時間が進み（最大8時間分）、帰還時に結果を受け取る形式。

- **プラットフォーム**: ブラウザ（GitHub Pages、バックエンドなし）
- **保存**: localStorage（チェックサムによる改ざん検出付き）
- **プレイスタイル**: 出発前にパーティ・スタンスを設定し、あとは放置。戦闘中にスタンスやターゲットを手動変更可能。

### ゲームの流れ

```
パーティ編成 → ダンジョン選択 → 出発 → 自動探索（放置）
     ↓                                        ↓
帰還・報酬受取 ←── 踏破 / 全滅 / 手動撤退 ───┘
```

### 2-3. ステータス計算式

```
maxHp   = 基本HP   + (Lv-1) × HP/Lv   + 装備ボーナス
attack  = 基本ATK  + (Lv-1) × ATK/Lv  + 装備ボーナス
defense = 基本DEF  + (Lv-1) × DEF/Lv  + 装備ボーナス
speed   = 基本速度 + floor((Lv-1) × 速/Lv) + 装備ボーナス
maxMp   = 基本MP（レベルでは増加しない）
```

MPはレベルアップで増加しない（固定値）。装備ボーナスで補う設計。

### 2-4. 経験値・レベルアップ

```
次のレベルに必要な経験値 = 現Lv × 現Lv × 15
（例: Lv1→2=15, Lv5→6=375, Lv10→11=1500）
```

- レベルアップ時は HP・MP・各ステータス自動更新
- 探索終了時（成功・撤退のみ）に経験値付与。全滅時は経験値なし
- 経験値はパーティ人数で均等分配

### 2-5. 速度と行動間隔

```
行動間隔(ms) = 60000 / speed
（例: speed=30 → 2000ms間隔、speed=60 → 1000ms間隔）
```

## 3. スキル

### 3-1. 概要

- スキルは職業ごとに **レベルアップで自動習得**（デッキ構築なし）
- 習得レベル: 1, 5, 10, 15, 20, 30, 40, 55, 70, **99**
- 各キャラ最大3スロットを「アクティブスキル」として選択可能
- 戦闘中はクールダウンと発動条件を満たすと自動使用
- **Lv99（奥義）**: 超強力だが副作用・制約あり。90s以上のCDと高MPコスト

### 3-2. スキルタイプ

| タイプ | 内容 |
|--------|------|
| attack | 敵へのダメージ（物理ベース） |
| special | 高倍率や全体攻撃などの強力技 |
| heal | HP回復・状態異常回復 |
| buff | 味方へのバフ付与 |
| debuff | 敵への状態異常付与 |

### 3-3. 自動発動条件

| 条件 | 発動タイミング |
|------|--------------|
| always | 常時（スキル優先） |
| hp_below_50 | 自分のHP50%未満 |
| hp_below_25 | 自分のHP25%未満 |
| combat_start | 戦闘開始から5秒以内 |
| enemy_hp_above_75 | 敵のいずれかがHP75%以上 |

| 混沌の靴 | 装飾 | - | - | - | - | +25 | - | 混沌の速さで駆ける靴 |
| 虚無の大盾 | 装飾 | - | - | +15 | +50 | - | 闇・光耐性60%・自動復活1回 | 光と闇を60%軽減、1度だけ戦闘不能を防ぐ究極の盾 |

### 4-4. 強化システム

武器・防具・アクセサリは最大 **+10** まで強化可能。

```
強化後ステータス = 基本値 + 強化Lv × ceil(基本値 × 0.2)
（例: ATK+10の装備を+5強化 → 10 + 5×2 = +20）
```

| 強化段階 | 成功率 | 費用(ゴールド) | 破損率 |
|---------|--------|--------------|--------|
| +0→+1 | 95% | 100 | 10% |
| +1→+2 | 88% | 250 | 10% |
| +2→+3 | 78% | 500 | 10% |
| +3→+4 | 68% | 1,000 | 10% |
| +4→+5 | 55% | 2,000 | 10% |
| +5→+6 | 42% | 4,000 | 10% |
| +6→+7 | 30% | 8,000 | 10% |
| +7→+8 | 18% | 20,000 | 10% |
| +8→+9 | 9% | 50,000 | 10% |
| +9→+10 | 3% | 100,000 | 10% |

- 失敗しても破損しない場合はゴールドのみ消費
- 破損時は装備が消滅（ゴールドも消費）

### 4-5. 属性武器・属性耐性

**属性武器**: 武器に `element` を設定することで通常攻撃に属性ダメージが加わる。属性相性（セクション11）が適用される。

**属性耐性**: 装備の `elementResist` で特定属性のダメージを軽減できる。複数装備の耐性は加算（最大90%）。

```
最終ダメージ = ダメージ × (1 - 耐性値)
（例: 火耐性50% → 火ダメージが50%軽減）
```

### 4-7. 装備レアリティ

装備品は **common（一般）** と **rare（レア）** の2段階のレアリティを持つ。

| レアリティ | 判定条件 | ドロップ確率 |
|-----------|---------|------------|
| rare | bossOnly / 特殊効果あり / 属性付き のいずれか | 10% |
| common | 上記以外 | 90% |

ドロップ時にレアリティを抽選し、該当ティアの装備品プールから選出される。

## 5. パーティ編成

### 5-1. 基本構成

- パーティは最大4名
- 探索前に各キャラの装備・スキルスロット・スタンスを設定

### 5-2. マイパーティセット

3スロット分のパーティ設定を保存できる。各セットには以下を記録：
- キャラクター選択
- 装備構成（スロットごと）
- アクティブスキルの選択（最大3つ）
- スタンス

ロード時にキャラクターのステータスは現在のレベルに基づき自動再計算される。
すでに別のキャラが装備しているアイテムは自動的に付け替えられる。

### 5-3. 帰還後の処理

探索終了後（踏破・撤退・全滅）にパーティを帰還させると：
- パーティ全員のHP・MPが全回復
- レベルアップしたキャラのステータスが更新
- 新しいスキルが自動習得

## 8. 戦闘システム

### 8-1. 基本的な流れ

戦闘はターン制シミュレーション。1ターン = 1秒のゲーム内時間。

```
各ターン: 全生存ユニット（味方＋敵）がそれぞれ1回ずつ行動する
行動順: 素早さ（speed）が高い順。同値はパーティ→敵の順
全員が行動し終えたら次のターンへ

行動間隔（ゲーム内秒） = 1ターン/1秒
速度倍率: ×1（デフォルト）/ ×2（永遠の聖域クリア後）/ ×3（混沌の試練クリア後）
```

速度倍率はオフライン放置にも適用される。

### 8-2. 素早さの役割

- **行動順決定のみ**（1ターン1行動は全員共通）
- spd_up バフ: 有効速度が上昇し、行動順が前になる
- bind（束縛）デバフ: 有効速度が低下し、行動順が後ろになる

### 8-2. 通常攻撃のダメージ計算

```
基本ダメージ = max(1, round(ATK × ATK補正 × last_stand補正) - round(DEF × DEF補正))
属性補正   = 武器属性がある場合、属性相性テーブル × element_boost装備補正
耐性補正   = 1 - 対象のelementResist（最大90%）
最終ダメージ = 基本ダメージ × 属性補正 × 耐性補正 × 分散(85〜115%) × 天機補正
天機(クリティカル): 1% + crit_chance装備ボーナス の確率で×2倍
吸血: ダメージ × lifesteal率 をHP回復（自身生存時のみ）
```

### 8-3. スキルダメージ計算

```
物理スキル: max(1, (ceil(ATK × 倍率 × スキルダメ補正 × ATKバフ × フォーカス × last_stand補正)
                     - DEF補正) × 分散 × 天機 × 属性相性 × element_boost × 耐性補正)
魔法スキル: 同上ただしDEFを半分として計算
回復スキル: 基本回復量 × (1 + healing_boost装備補正)
```

### 8-4. 特殊ルール

- **天機（クリティカル）**: 1%の確率でダメージ2倍。ログに「天機！」と表示
- **結界**: 味方がこのバフを持つ場合、次の1回のダメージを無効化して消滅
- **集中**: 攻撃・特殊スキル使用時にダメージ×2、使用後消滅
- **再生**: 行動のたびにMaxHP5%を回復（4行動持続）
- **腐毒（敵）**: 敵が行動するたびにMaxHPの3%ダメージ

### 8-5. ターゲット

- デフォルト: 先頭の生存敵を自動選択
- プレイヤーが手動でターゲット指定可能（同じ敵を再クリックで解除）
- ターゲット指定はパーティ全員に適用

### 8-6. MP回復

- 戦闘終了→次フロアへ移動する際に **MaxMPの15%** × スタンス補正 を自動回復
- 帰還時にHP・MP全回復

## 10. バフ・状態異常

### 10-1. バフ一覧（味方）

| バフ名 | 効果 | 持続 | 最大スタック | スタック効果 |
|--------|------|------|------------|------------|
| 攻↑ (atk_up) | ATK上昇 | 3行動 | 3 | ×1.25 / ×1.5 / ×1.7 |
| 守↑ (def_up) | DEF上昇 | 3行動 | 3 | ×1.25 / ×1.5 / ×1.7 |
| 速↑ (spd_up) | 行動間隔短縮 | 3行動 | 3 | ×0.75 / ×0.60 / ×0.50 |
| 再生 (regen) | 毎行動MaxHP5%回復 | 4行動 | 1 | - |
| 結界 (barrier) | 次のダメージ1回無効 | 50行動 | 1 | - |
| 集中 (focus) | 次の攻撃スキル×2 | 50行動 | 1 | - |
| 加護 (guard) | 防御力強化（守↑と同系統の保護効果） | 50行動 | 1 | - |

### 10-2. 状態異常一覧（敵への付与）

| 状態異常 | 効果 | 持続 | 付与確率 | スタック |
|---------|------|------|---------|---------|
| 腐毒 (poison) | 行動時にMaxHPの3%ダメージ | 4行動 | 55% | なし |
| 硬直 (stagger) | 1回行動スキップ | 1行動 | 40% | なし |
| 束縛 (bind) | 行動間隔延長（×1.5/2.0/2.5） | 3行動 | 50% | 3段 |
| 封力 (seal_atk) | ATK低下（×0.70/0.50/0.35） | 4行動 | 55% | 3段 |
| 封守 (seal_def) | DEF低下（×0.70/0.50/0.35） | 4行動 | 55% | 3段 |

### 10-3. 状態異常一覧（敵からプレイヤーへ）

ボスや一部の敵が特殊能力として攻撃時にプレイヤーへデバフを付与する。

- **封力付与**: ATKが低下
- **封守付与**: DEFが低下
- **束縛付与**: 行動間隔が延長

**バフによる相殺**: 対応するバフを持っている場合、デバフは吸収されてバフが1段階低下する。
- 封力 → 攻↑ バフを1段階消費
- 封守 → 守↑ バフを1段階消費
- 束縛 → 速↑ バフを1段階消費

### 10-4. バフによるデバフ解除

バフスキルには対応する状態異常を解除する効果もある：
- 攻↑付与 → 封力を解除
- 守↑付与 → 封守を解除
- 速↑付与 → 束縛を解除

## 11. 属性相性

6属性: 無・火・氷・雷・光・闇

### 火・氷・雷の三竦み

```
火 → 雷が弱点（×0.5）
氷 → 火が弱点（×0.5）
雷 → 氷が弱点（×0.5）
```

### 光・闇

| 攻撃 \ 防御 | 光 | 闇 |
|------------|----|----|
| 光 | ×0.5（耐性） | ×1.5（弱点） |
| 闇 | ×1.5（弱点） | ×0.5（耐性） |

属性のない攻撃（無属性）は相性補正なし（×1.0）。

### 属性耐性（装備）

装備の `elementResist` プロパティで特定属性への耐性を持てる。耐性は加算合成（複数装備）で最大0.9（90%）にクランプ。耐性は属性相性補正の後に適用される。

## 12. 報酬システム

### 12-1. 戦闘報酬

- 敵撃破でゴールドとEXPを獲得（フロアの全敵分を加算）
- 通常戦闘勝利時 **5%の確率** で装備品ドロップ（難易度に応じたティア）
- ボス撃破時 **30%の確率** で装備品ドロップ
- イベントフロアでも装備品・ゴールドを入手可能

### 12-2. 帰還時の報酬処理

| 探索結果 | ゴールド | EXP | 装備品 |
|---------|---------|-----|--------|
| 踏破 | 全額獲得 | 全額獲得 | 全て獲得 |
| 撤退 | 全額獲得 | 全額獲得 | 全て獲得 |
| 全滅 | 0（ペナルティあり） | 0 | なし |

**全滅ペナルティ**: 現在所持ゴールドの50%を失う

### 12-3. EXP分配

```
各メンバーの獲得EXP = 累計EXP ÷ パーティ人数（端数切り捨て）
```

### 12-4. 装備品の上限

所持できる装備品の最大数は **300個**。超過した分は新規入手時に古いものが溢れる（売却・整理推奨）。

### 12-5. 装備品の売却

| 強化段階 | 売却価格の算出 |
|---------|--------------|
| +0 | 50G |
| +N | 50G + 強化1〜N段のコスト合計×20% |

| 装備強化値 | 0〜10 |
